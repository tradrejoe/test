                                              Lazar Ljubenovic kinesias 4 years ago

                                              Using promises is easier when you're embracing a more functional style. For example, you can do `promise.then(handleData)` instead of `async function() { const res = await promise ; return await handleData(res) }`. It's not so black and white. `async`/`await` is NOT there to _replace_ promises, it's to provide an alternative for handling async code.
